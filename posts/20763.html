数据库基础语法总结

发表于2021-12-02|更新于2022-01-22|学习
|字数总计:1.9k|阅读时长:6分钟|阅读量:

数据库和表的基本操作

连接数据库，输入命令:mysql -u用户名 -p密码

注意：命令结尾要加英文状态的分号;

1.创建数据库

语法：create database 数据库名;

查看MySQL中已经存在的数据库,命令：show databases;

选择数据库，命令：use 数据库名;

注：数据库名区分大小写，同样表中的字段也区分

2.创建表

语法：

create table 表名
(
    字段名1 数据类型,
    字段名2 数据类型
);

查看表的基本结构，命令：describe 表名; 简写为desc 表名;

查看数据库存在的表，命令：show tables;

查看表的详细结构，命令：show create table 表名;

3.使用主键约束

添加主键约束，有以下两种方式：

  • 在定义列的同时指定主键

    create table 表名
(
    字段名1 数据类型 primary key,
    字段名2 数据类型
);

  • 在定义完字段后再指定主键

    create table 表名
(
    字段名1 数据类型,
    字段名2 数据类型,
    primary key(字段名1)
);

两者有区别，前者不赋初值，后者赋值为0

也可以使用联合主键,即 primay key(字段名1，字段名2);

注：表格按主键排序

4.外键约束

语法:constraint 外键名 foreign key 字段名 references 主表名(主键名)

例子:给员工表(t_emp)添加外键；

CREATE TABLE t_dept
(
    deptId INT PRIMARY KEY,
    name VARCHAR(22),
    location VARCHAR(50)
);

CREATE TABLE t_emp
(
    id INT PRIMARY KEY,
    name VARCHAR(22),
    deptId INT,
    CONSTRAINT fk_emp_dept1 FOREIGN KEY(deptId) REFERENCES t_dept(deptId)
);

deptId是t_emp的外键

注:已经是外键的列不能再设置为主键

  在设置的外键的表内，也要加上原主键字段及其数据类型

5.添加常用约束

  • 唯一约束

    Unique ，要求该列唯一，允许为空，但只能有一个空值

    例如：name VARCHAR(22) UNIQUE

  • 非空约束

    NOT NULL

    例如： name VARCHAR(22) NOT NULL,

  • 默认约束

    DEFAULT;给字段设置一个默认值

    例如：sex VARCHAR(2) DEFAULT '男'

  • 设置表的属性值自动增加

    auto_increment;

    例如: id int PRIMARY KEY AUTO_INCREMENT,

    注意:

    • 如果是添加字符串型默认值要使用单引号，如果是整型则不需要加任何符号；

    • 如果要添加的是中文默认值，则需要加上DEFAULT CHARSET=utf8;使用英文字符则不需要。

      例如：

      CREATE TABLE t_emp(
    id INT PRIMARY KEY,
    name VARCHAR(22),
    sex VARCHAR(2) DEFAULT '男'
) DEFAULT CHARSET=utf8;

6.修改表名

语法：alter table 旧表名 rename 新表名

7.修改字段名及字段数据类型

修改字段名

语法：alter table 表名 change 旧字段名 新字段名 新数据类型

例如：alter table mall_products change prod_id ID varchar(20);

注:如果不需要修改字段的数据类型,可以把新字段的数据类型设置为和原来一样但不要空着

修改字段数据类型

语法规则：alter table 表名 modify 字段名 数据类型

8.添加删除字段

添加字段

语法：alter table 表名 add 新字段名 数据类型[约束条件][FIRST|AFTER] 已存在字段名;

若不做[FIRST|AFTER]的位置说明,默认放在表的最后一列.

  • 在表的最后一列添加字段，ALTER TABLE Mall_products ADD prod_country varchar(30);
  • 在表的第一列添加字段，ALTER TABLE Mall_products ADD prod_country varchar(30) FIRST;
  • 在表的指定列后添加字段，ALTER TABLE Mall_products ADD prod_country varchar(30) AFTER prod_name;
删除字段

语法：alter table 表名 drop字段名;

例如：alter table mall_products drop prod_price;

9.修改字段的排列位置

语法：alter table 表名 modify 字段1 数据类型 FIRST|AFTER 字段名2

  • 修改字段为表的第一个字段，ALTER TABLE Mall_products MODIFY prod_price FLOAT FIRST;
  • 修改字段到表的指定列之后，ALTER TABLE Mall_products MODIFY prod_price FLOAT AFTER prod_country;

10.删除表的外键约束

语法：alter table 表名 drop foreign key 外键约束名

这个外建名，就是添加约束时的一个外键名

11.数据操作

（1）插入数据

语法：insert into 表名(字段名) values(内容)

例如：insert into mall_products(id,country_id) values(1,86);

也可以插入多条记录,语法：INSERT INTO 表名 (字段名) VALUES (内容1), (内容2), (内容3);

（2）更新数据

语法：UPDATE 表名 SET 字段名1 = 内容1, 字段名2 = 内容2, 字段名3 = 内容3 WHERE 过滤条件;

例如：update mall_products set country_name='Pakistan',country_id=92 where id = 2;

修改id为2的元组内容

（3）删除数据

语法：delete from 表名 where 条件语句;

例如：delete from mall_product where id =2 or id =3; 同时删除两行。

delete from 表名; 删除表中所有行

truncate table 表名; 删除的是表

12.SELECT查询

(1)查询数据表中字段的内容

  • 查询数据表中指定字段的内容

    语法格式：SELECT 字段名 FROM 表名;
    当同时查询”多个字段”下的内容时，只需要在字段之间加入逗号。
    语法格式：SELECT 字段名1,字段名2 FROM 表名;

  • 查询数据表中的所有内容

    SELECT * FROM 表名;

小提示： MySQL 中的 SQL 语句对关键字的大小写不敏感， SELECT 和 select 是一样的。 但是许多开发者更喜欢将关键字大写，表名和字段名使用小写，养成这种习惯，以后写出来的脚本就更容易阅读和维护了。

(2)带IN关键字条件查询

语法格式：SELECT 字段名 FORM 表名 WHERE 字段名 IN(n1,n2,n3,...);
等价于 SELECT 字段名 FROM 表名 WHERE 字段名 = n1 OR 字段名 = n2 OR 字段名……;

(3)带NOT IN关键字查询

语法格式：SELECT 字段名 FROM 表名 WHERE 字段名 NOT IN(n1,n2,n3,...);

(4)带BETWEEN AND的范围查询

语法格式：SELECT 字段名 FROM 表名 WHERE 字段名 BETWEEN n1 AND n2;

(5)带NOT BETWEEN AND关键字的查询

语法格式：SELECT 字段名 FROM 表名 WHERE 字段名 NOT BETWEEN n1 AND n2;

(6)带LIKE的字符匹配查询

语法格式：SELECT 字段名 FROM 表名 WHERE 字段名 LIKE '字符%'; 其中 % 的位置可以根据需要在字符间变化。
B开头的字符，用B%表示，B在中间，用%B%表示。
注：对于%B%,通配符%告诉MySQL，返回字符串中间带B字符的记录，不管B前面和后面分别有多少字符。
也可以使用通配符_模糊匹配数据内容，但下划线通配符_只能模糊匹配1个字符，如果要匹配多个字符，要多次使用。

(7)去除重复结果

语法格式：SELECT DISTINCT 字段名 FROM 表名;

(8) 带AND与OR的多条件查询

  • AND

    语法格式：SELECT 字段名 FROM 表名 WHERE 表达式1 AND 表达式2;
    例如：SELECT * FROM Mall_products WHERE prod_name LIKE ‘%toy%’ AND prod_price > 5;

  • OR

    语法格式：SELECT 字段名 FROM 表名 WHERE 表达式1 OR 表达式2;
    注: OR 可以和 AND 一起使用。但是 AND 的优先级要高于 OR 的优先级.
    推荐使用 IN 关键字.

(9)对查询结果排序

语法格式：SELECT 字段名 FROM 表名 ORDER BY 字段名 [ASC[DESC]];
ASC 升序关键字
DESC 降序关键字

(10)分组查询

语法格式：SELECT 字段名 FROM 表名 GROUP BY 字段名;
注：查询的是每个分组中 首次出现的一条记录。

(11)用LIMIT限制查询结果数量

语法格式：SELECT 字段名 FROM 表名 LIMIT [OFFSET,] 记录数;
参数说明：
第一个参数，OFFSET，可选参数，表示偏移量，如果不指定默认值为0，表示从查询结果的第一条记录开始，若偏移量为1，则从查询结果中的第二条记录开始，以此类推。
第二个参数，记录数，表示返回查询结果的条数。
注：order by 放在限制查询语句之前，选择条件结果之后。
